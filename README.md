# Caesar Cipher Tool

This is a simple Caesar Cipher tool with a graphical user interface (GUI) built using Python and `tkinter`.

## How to Use

1. Enter the text you want to encrypt or decrypt.
2. Enter the shift value (number of positions to shift).
3. Click "Encrypt" to see the encrypted text or "Decrypt" to see the decrypted text.

## Project Structure

- `caesar_cipher.py`: Contains the Caesar Cipher logic.
- `main.py`: Contains the GUI code.
- `README.md`: This file with project details.
